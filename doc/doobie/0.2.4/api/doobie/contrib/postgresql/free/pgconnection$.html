object pgconnection

Algebra and free monad for primitive operations over a org.postgresql.PGConnection. This is a low-level API that exposes lifecycle-managed JDBC objects directly and is intended mainly for library developers. End users will prefer a safer, higher-level API such as that provided in the doobie.contrib.postgresql.hi package.

PGConnectionIO is a free monad that must be run via an interpreter, most commonly via natural transformation of its underlying algebra PGConnectionOp to another monad via Free.runFC.

The library provides a natural transformation to Kleisli[M, PGConnection, A] for any exception-trapping (Catchable) and effect-capturing (Capture) monad M. Such evidence is provided for Task, IO, and stdlib Future; and transK[M] is provided as syntax.

// An action to run
val a: PGConnectionIO[Foo] = ...

// A JDBC object
val s: PGConnection = ...

// Unfolding into a Task
val ta: Task[A] = a.transK[Task].run(s)
